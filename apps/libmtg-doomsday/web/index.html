Doomsday

Simulation tools for the Legacy Doomsday deck, built on the libmtg Magic: The Gathering engine.

Goldfish →

Monte-Carlo a decklist against a passive opponent: how fast does it cast Doomsday, and how much protection is in hand?

Hand explorer →

Deal opening hands as cards; see the learned model's instant read (P(cast), E[TTD], keep/mull) and reveal the simulated truth for any hand.

Pile builder →

Generate a board state to the moment Doomsday resolves, arrange the five-card pile, and share it as a URL.

Writeups →

Worked analyses built on the goldfish sim.
