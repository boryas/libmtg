Maps CR rules and engine mechanics to the cards that use them.
Given a rule or mechanic, find which cards exercise it and how.
Identifies non-standard patterns.

For engine architecture and primitive vocabulary, see [[file:DESIGN.org][DESIGN.org]].

*Maintain this file*: when adding or modifying a card, update the relevant
sections below. When adding a new engine primitive, add a section for it.

* How to Read

- *Standard* — uses existing engine primitives, no special logic.
- *Notable* — correct but uses an unusual pattern worth knowing about.
- *Abnormal* — workarounds, hardcoded shortcuts, or missing features.

* Mechanic → IR Decomposition

primitives. *Effect context* = the verb appears as a spell/ability effect;
*Cost context* = the verb appears as a cost component.

* Mana Production (CR 106)

** Tap for mana (basic pattern)

All basic lands, ABU duals, Great Furnace use ~tap_produces(mana_string)~.

| Card                    | Produces     | Notes                       |
|-------------------------+--------------+-----------------------------|
| Basic lands (6)         | Single color | ~basic_land()~ helper       |
| Snow-Covered basics (6) | Single color | ~snow_basic()~ helper       |
| ABU duals (10)          | Two colors   | Two ~tap_produces~ calls    |
| Great Furnace           | {R}          | Also typed as Artifact      |

*Design gap (2026-05-10):* basic-land mana abilities are currently
authored on each card via ~ir_tap_mana(color)~. The CR-correct model
(CR 305.6) is that the ability is granted by the *basic land subtype*
itself — Island grants ~{T}: Add U~, Swamp grants ~{T}: Add B~, etc.
Underground Sea has both abilities because it has both subtypes. The
right implementation is a single continuous effect keyed on subtype,
not 50+ ~ir_tap_mana~ calls. Doing this would also make Urborg
(grants Swamp subtype to all lands) and Blood Moon / Magus of the
Moon (replace subtypes with Mountain) fall out for free instead of
being special-cased. *Deferred* — finishing cost-IR migration is
prioritized first; this is its own architectural pass.
